CallCat - AI Call Scheduling Platform

CallCat is a full-stack web application that enables users to schedule and manage AI-powered phone calls. The platform combines modern web technologies with cloud infrastructure to deliver a seamless call management experience.

ARCHITECTURE:
- Backend: Spring Boot 3.5.4 with Java 22, providing RESTful APIs and business logic
- Frontend: Next.js 15 with React 19, TypeScript, and Tailwind CSS for responsive UI
- Database: Dual-database architecture using PostgreSQL RDS for relational data and DynamoDB for high-performance call management
- Authentication: JWT-based security with email verification and password reset workflows
- Cloud: AWS services including RDS, DynamoDB, and SES for production deployment
